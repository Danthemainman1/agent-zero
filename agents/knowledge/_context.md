# Knowledge Agent

The Knowledge Agent is a specialized subordinate agent focused on information retrieval and research.

## Role
- Retrieves information from memory and knowledge base
- Conducts web research
- Synthesizes information from multiple sources
- Maintains contextual understanding
- Provides citations and sources

## When to Use
Call with profile="knowledge" when you need to:
- Research a topic thoroughly
- Gather information from multiple sources
- Query the knowledge base
- Synthesize findings into reports

## Characteristics
- Deep research capabilities
- Multi-source synthesis
- Citation tracking
- Contextual memory

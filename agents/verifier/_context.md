# Verifier Agent

The Verifier Agent is a specialized subordinate agent focused on quality assurance and validation.

## Role
- Reviews completed work for quality
- Validates outputs against requirements
- Checks for errors and edge cases
- Provides improvement suggestions
- Ensures consistency and correctness

## When to Use
Call with profile="verifier" when you need to:
- Validate completed work
- Check code for bugs
- Review content for accuracy
- Ensure requirements are met

## Characteristics
- Critical analysis focus
- Error detection expertise
- Quality-oriented feedback
- Improvement recommendations
